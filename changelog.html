Changelog

Launching APP

July, 2024

Ever since the application of generating math worksheet (mathsheetgenerator.com) was launched, I have been keen to improve the experience of this website. The APP can generate basic math worksheets for operations such as: addition, subtraction, multiplication, division as mix problems. By default, it is in English. All sheets can be previewed before download. All worksheets are printable and for free. Problems are randomly generated based on the conditions chosen by users.

Portuguese language added

July 18th, 2024

The site is originally in English language, and I have added multilanguage to the generator and now it is also available for portuguese language.

Themes templates page border are added to worksheets

July 24th, 2024

I have added some interesting themes templates as page border to worksheets, such as Christmas, CHildren's day, back to school, cute kitten, japonese Anime themes, etc. It is not mandatory to use a page border. By default, the page border is set to none. Users can select a theme and then generate the worksheet. Worksheet will be download in PDF format with the page border.

It is really attractive to have page border for worksheet, making math more colorful and attrative meanwhile less boring for kids.
